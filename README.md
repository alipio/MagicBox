MagicBox
========

All about configuration of programming tools, MacOSX/Linux system.

 * MacOSX
   * bash
   * iterm2
 * eclipse
   * eclipse preference.
 * git
   * A script to setup git configuration.
   * git bash completion.
   * git ignore.
 * moinmion
   * A script to upgrade and configure moinmoin installation.
 * python
   * nosetest config
   * pylint config
   * pep8 config
 * emacs
   * emacs-gui
   * emacs-macosx-terminal
 * vim
   * vimrc
 * sublime
   * config
